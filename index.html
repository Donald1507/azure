Les services Microsoft Azure

Lancée le 1er Février 2010, Azure est la plateforme Cloud de Microsoft.
Elle regroupe divers services de Cloud Computing.
On y trouve à la fois des solutions open-source que des technologies propriétaires de Microsoft ou d’autres entreprises. A l’origine, la plateforme s’appelait Windows Azure.
Elle fut renommée Microsoft Azure en 2014.

Microsoft Azure : à quoi ça sert ?

Comme les autres fournisseurs de services Cloud, Microsoft Azure permet:

  • de profiter de ressources de Cloud Computing à la demande.
  • aux entreprises de faire d’importantes économies en leur évitant d’avoir à ériger un centre de données sur site, de le maintenir, de le mettre à jour, de le refroidir et de payer l’électricité.

Microsoft Azure: quels sont les différents services?

(Voir les services)

Microsoft Azure propose plus d’une vingtaine de catégories de services différentes. Outre les services de:

  • calcul,
  • stockage,
  • développement web et mobile,
  • d’intégration,
  • networking,
  • conteneurisation,
  • d’analyse de données.

Comment Azure fonctionne t-il?


Azure offre une grande collection de services, y compris
la plateforme en tant que service (PaaS),
l'infrastructure en tant que service (IaaS),
la base de données gérée en tant que service (DBaaS).

Azure est tout simplement une vaste collection de serveurs et de matériel réseau qui exécutent un ensemble complexe d’applications distribuées qui orchestrent la configuration et l’opération du matériel et des logiciels virtualisés sur ces serveurs.
C’est cette orchestration qui fait toute la puissance d’Azure. Les utilisateurs ne sont plus responsables du suivi, ni de la mise à niveau du matériel, Azure effectuant tout cela en arrière-plan.

Microsoft Azure: quels sont les services pour l'Intelligence Artificielle?

  • Détecteur d’anomalies
    Ajoutez aisément des fonctionnalités de détection d’anomalies à vos applications

  • Recherche cognitive Azure
    Service de recherche cloud alimenté par l'intelligence artificielle pour le développement d'applications mobiles et web

  • Azure Databricks
    Plateforme d’analyse rapide, simple et collaborative basée sur Apache Spark

  • Azure Machine Learning
    Proposez l’intelligence artificielle à tous avec une plateforme de bout en bout, scalable et approuvée qui inclut l’expérimentation et la gestion des modèles

  • Azure Open Datasets
    Plateforme cloud permettant d'héberger et de partager des jeux de données ouverts et organisés afin d'accélérer le développement de modèles Machine Learning

  • Azure Cognitive Services
    Ajoutez des fonctionnalités d’API intelligentes pour obtenir des interactions contextuelles

  • Vision par ordinateur
    Dévoilez des informations pertinentes à partir d’images

  • Content Moderator
    Modération automatisée des images, textes et vidéos

  • Machines virtuelles Science des données
    Environnement préconfiguré enrichi pour le développement de l’intelligence artificielle

  • Visage
    Détectez, analysez, organisez et identifiez les visages dans vos photos

  • Form Recognizer
    Le service d’extraction de document optimisé par l’intelligence artificielle qui comprend vos formulaires

  • Lecteur immersif
    Donnez aux utilisateurs de tous âges et de toutes capacités les moyens de lire et de comprendre du texte

  • Kinect DK
    Créez des modèles vocaux et de vision par ordinateur à l'aide d'un kit de développement doté de capteurs IA avancés

  • Language Understanding
    Formez vos applications pour qu’elles comprennent les commandes de vos utilisateurs

  • Personalizer
    Un service d’IA qui offre une expérience utilisateur personnalisée

  • Traduction vocale
    Intégrez facilement la traduction vocale en temps réel à votre application

  • Analyse de texte
    Évaluez facilement les sentiments et les thèmes pour comprendre ce que les clients recherchent

  • Synthèse vocale
    Fonctionnalité du service Speech qui convertit le texte en langage réaliste

  • Traducteur
    Effectuez facilement une traduction automatique avec un simple appel d’API REST
